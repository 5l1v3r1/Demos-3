Mandelbrot Explorer

The Mandelbrot set is one of the most complex and beautiful mathematical objects known. Mandelbrot Explorer allows you to explore this often striking set using both touch and a traditional mouse. Additionally, a detailed explanation of the Mandelbrot set and how this application was implemented is available in How to explore the Mandelbrot set using HTML5.
